## **Burkina Faso : Répartition des 12 nouveaux arrondissements par quartiers de Ouagadougou**
Depuis les dernières élections couplées du 2 décembre 2012, Ouagadougou compte désormais 55 secteurs et 12 arrondissements au lieu de 5. Ce redécoupage a pour objectif de rapprocher l’Administration de l’administré.
[ ![](https://www.burkinatourism.com/local/cache-vignettes/L516xH291/ouagadougou-4872d.jpg?1751749648)](https://www.burkinatourism.com/IMG/jpg/ouagadougou.jpg)
**Arrondissement 1** : Bilbalogo ; Saint Léon ; Zangouettin ; Tiedpalogo ; Koulouba ; Kamsonghin ; Samandin ; Gounghin Sud ; Gandin ;  
Kouritenga ; Mankougoudou .
**Arrondissement 2** : Paspanga ; Ouidi ; Larlé ; Kologh Naba ; Dapoya 2 ; Nemnin ; Niogsin ; Hamdalaye ; gounghin Nord ; Baoghin .
**Arrondissement 3** : Camp militaire ; Naababpougo ; Kienbaoghin, Zongo ; koumdayonré ; nonsin ; rimkièta ; Tampouy ; Kilwin .
**Arrondissement 4** : Tanghin ; Sambin barrage ; Somgandé ; zone industrielle ; Nioko 2 ; Bendogo ; Toukin .
**Arrondissement 5** : Zogona ; Wemtenga ; Dagnoën ; Ronsin ; Kalgondin .
**Arrondissement 6** : Cissin ; kouritenga ; Pissy .
**Arrondissement 7** : Nagrin ; Yaoghin ; Sandogo ; Kankasin ; Boassa .
**Arrondissement 8** : Zaghtouli ; Zongo Nabitenga ; Sogpèlcé ; Bissighin ; Bassinko ; Dar-es-Salam ; Silmiougou ; Gantin .
**Arrondissement 9** : Bangpooré ; Larlé Wéogo ; Marcoussis ; Silmiyiri ; Wob Riguéré ; Ouapassi .
**Arrondissement 10** : Kossodo ; Wayalghin ; Godin ; Nioko 1 ; Dassosgho ; Taabtenga .
**Arrondissement 11** : Dassasgo ; yemtenga ; Karpala ; Balkuy ; Lanoayiri ; Dayongo ; Ouidtenga .
**Arrondissement 12** : Patte d’oie ; Ouaga 2000 ; trame d’accueil de Ouaga 2000 .
